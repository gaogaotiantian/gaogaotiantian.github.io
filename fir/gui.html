什么是GUI

GUI(Graphical User Interface)一般指可视化界面。大家平时玩五子棋很少有输入坐标玩的，这个GUI系统就是为了让大家可以更好的和AI交流

准备工作

GUI的全部内容都在git repo里了，所以你需要的准备工作只是从git里pull下来

$ git pull

GUI用的是gtk+3.0的库，所以你需要先下载库。不过Linux和mac的用户可以直接用我写好的脚本安装

$ ./installLibs.sh

在库下载完成以后我们就可以准备编译运行了

编译运行

我们依然用make来编译，只不过用一个不同的file，需要用-f来指定makefile。注意GUI版本和Terminal版本是共享header和ai文件的，只是有不同的主source file。

$ make -f makefile_gui

然后我们只需要运行编译好的程序即可：

$ ./gfir

GUI简单说明

GUI本身很简单，按那儿就下哪儿。只有两个按钮，“从头再来”是正常的开局，“随机开局”是棋盘上随机2白1黑之后黑棋开始下。
