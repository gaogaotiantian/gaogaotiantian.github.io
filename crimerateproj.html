项目细节

综述

本项目核心是为用户开发一款方便快捷的应用，可以输入城市名称，获取这个城市是否安全。包括城市的犯罪数量，犯罪率，和其他城市相比的情况等等数据。我们争取在网页端、iOS和Android全部实现。

数据结构

在这个project中，我们的数据将会保存在一个 json文件里。核心结构为一个dict，key是城市名称，data是有关城市的数据。这个json文件由数据组负责。

Example 

  
    {
      "Santa Barbara, CA":
      {
        "Number":363,
        "Rate":395,
        "Index":80,
        "Ranking":70,
      }
      "Los Angeles, CA":
      {
        ...
      }
    }

数据组职责

从FBI UCR的网站上爬下按照城市为单位的数据（可以用爬虫，也可以下载提供的xsl文件再用其他package读xls文件），原始数据保存人口和Violent Crime的数量。保存好原始数据之后，以原始数据为基础，做出整理后的数据。

  • Num: 城市总犯罪次数
  • Rate: 每10w人的犯罪率
  • Index: 犯罪指数，全美均值为100，80代表犯罪率是全美均值的80%
  • Ranking: 犯罪率排名，在这个城市生活比全美多少百分比的居民安全，最安全为100。如60代表这个城市的犯罪率要低于60%的美国居民所在的城市。
  • 你们自由发挥的空间

前端组职责

在自己负责的平台上，学习怎么导入json数据到自己的程序，学习如何引入一个搜索栏，用户只需要在搜索栏中打出自己的城市名称（实现Auto Complete即智能搜索），就可以显示这个城市的数据。数据已经由json文件给出，你需要在你的平台上用你觉得最美观的方法展示出来。 在数据组做出json文件之前，你们可以先用这个文件作为测试文件。在数据组做出json数据之后，请把他们的数据放到本地使用。
