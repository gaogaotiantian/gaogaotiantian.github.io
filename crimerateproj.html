项目细节

综述

本项目核心是为用户开发一款方便快捷的应用，可以输入城市名称，获取这个城市是 否安全。包括城市的犯罪数量，犯罪率，和其他城市相比的情况等等数据。我们争取 在网页端、iOS和Android全部实现。

数据结构

在这个project中，我们的数据将会保存在一个 json文件里。核心结构为一个 dict，key是城市名称，data是有关城市的数据。这个json文件由数据组负责。

Example 

  
    {
      "Santa Barbara, CA":
      {
        "Number":363,
        "Rate":395,
        "Index":80,
        "City Ranking":70,
        "Resident Ranking":80
      }
      "Los Angeles, CA":
      {
        ...
      }
    }

数据组职责

从FBI UCR的网站上爬下按照城市为单位的数据，原始数据保存人口和Violent Crime的 数量。保存好原始数据之后，以原始数据为基础，做出整理后的数据。

  • Rate: 每10w人的犯罪率
  • Index: 犯罪指数，全美均值为100，80代表犯罪率是全美均值的80%
  • City Ranking: 城市排名，犯罪总数量击败了多少比例的城市（越高越安全）
  • Resident Ranking: 居民排名，在这个城市生活比多少比例的居民安全
